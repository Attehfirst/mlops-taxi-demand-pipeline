# End-to-End MLOps Taxi Demand Prediction

## Overview
This project implements a production-grade MLOps pipeline for predicting taxi demand
using NYC Yellow Taxi trip data.

The system covers:
- Data ingestion
- Experiment tracking with MLflow
- Model training and evaluation
- Containerization with Docker
- Model serving (FastAPI)
- Monitoring

This project is based on the DataTalksClub MLOps Zoomcamp curriculum and extended
with custom structure and implementation.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- MLflow
- Docker
- FastAPI
